# TerraHeritage · 万物本草

**全球生命文化 3D 数字地球 —— 开源公益教育原型。**
一条可探索的知识链:**地点 → 生态 → 季节 → 植物 → 民族 → 传统医学 → 现代证据 → 安全**。

## 这是什么

万物本草不是普通百科,而是一个可旋转、可缩放、可深入探索的 3D 地球:
从全球视角进入大洲、国家、地区,直到一株植物、一个民族、一套传统医学体系。
所有事实都带来源标签与审核徽章——**无来源,不发布**。

## MVP 功能

- 🌍 真实球形 3D 地球:星空、大气光晕、海面日光、自动缓转、拖拽缩放、随高度缩放的阻尼手感
- 🗺️ 241 个国家/地区板块(Natural Earth 公有领域数据):悬停高亮、单击信息面板、双击飞近
- 🌱 季节引擎:南北半球相反、热带雨旱季、按月地表调色与高纬积雪、全年播放
- 🌿 植物知识卡**双层展示**:文化历史视角 × 现代证据视角(A–E 证据徽章)、安全提示、敏感物种坐标模糊
- 🧑‍🤝‍🧑 民族文化与医学文化图层,医学视图顶部固定科普免责声明
- 🐫 历史传播示意路线(丝绸之路、茶马古道等)大圆弧线动画,标注"历史推测示意"
- 🔍 全局搜索(⌘K):国家/植物/民族/医学体系,中英文皆可
- 🌐 中英双语、移动端底部抽屉、键盘快捷键、减少动态效果、三档画质
- 🔎 每条数据可打开来源抽屉;全部内容标注"示例数据·待审核"

## 快速开始

```bash
npm install
npm run dev        # http://localhost:3000
npm run build      # 生产构建 → dist/
npm run preview    # 本地预览构建产物
```

需要 Node.js 20+。可选配置:复制 `.env.example` 为 `.env`。

**快捷键**:空格 自转开关 · Esc 关闭面板 · R 回全球 · F 聚焦选中国家 · ⌘K/Ctrl+K 搜索 · L 图层

## 技术栈

Vite 7 · React 19 · TypeScript(strict)· Three.js + React Three Fiber + drei · zustand · Tailwind CSS · shadcn/ui · cmdk

无后端:内容全部以 TS 数据模块内置(`src/data/`),地理数据已内置(`src/data/geo/`),
运行时不依赖任何外部 API。目录结构与设计规格见 [docs/DESIGN.md](docs/DESIGN.md)。

## 数据、伦理与合规

本项目为**教育原型**:`src/data/` 内容均为**示例数据,全部标记 `draft`(待审核)**,
用于演示信息架构,不构成医疗、植物鉴定或文化定义依据。

- 数据关联的来源机构(POWO/Kew、GBIF、WHO、UNESCO、IUCN、中国植物志、BHL、WFO、
  NIH NCCIH、《中国药典》、EMA、PubMed)均为真实公开机构,引用仅指向机构主页。
- **不提供医疗建议**:仅作文化、历史与植物学教育;无剂量、无制备步骤、无疗效承诺、无购买引导。
- 传统使用 ≠ 现代临床证据;每种植物标注证据等级。
- 敏感物种(雪莲、冬虫夏草等)坐标模糊到生态区级,并明确提示不可随意采集。
- 民族文化呈现含现代生活,禁用猎奇化措辞;国界线不代表民族与文化边界。
- 完整治理模型(来源 S/A/B/C 分级、审核流程、FAIR/CARE 原则)见 [NOTICE.md](NOTICE.md) 与 [docs/DESIGN.md](docs/DESIGN.md)。
